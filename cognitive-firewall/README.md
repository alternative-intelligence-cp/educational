# Cognitive Firewall

**An AI Security Game for Learning Through Competition**

## What Is This?

Cognitive Firewall is a team-based game that teaches AI security concepts through competitive play. Players learn how adversarial attacks work (and how to defend against them) by experiencing them firsthand.

Think Team Fortress meets AI safety education.

## Why?

People who would never read a security whitepaper will spend hours perfecting their game strategy. By the time they're good at the game, they've internalized:
- Pattern recognition for adversarial prompts
- Social engineering tactics
- Boundary enforcement
- Trust calibration
- Defense-in-depth strategies

**It's stealth education through competition.**
